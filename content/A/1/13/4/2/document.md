###### A.1.13.4.2 - Requirement For Executor Agent Accord [Core]

Executor Agents must partner with Prime Agents through a formal Executor Agent Accord. An Executor Agent may not carry out any executive functions for a particular Prime Agent without a formal Executor Agent Accord.
