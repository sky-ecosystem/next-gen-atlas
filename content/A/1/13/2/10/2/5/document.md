###### A.1.13.2.10.2.5 - Remediation By Agent [Core]

The Agent must submit a Root Edit Proposal to update its Artifact to address the deficiencies identified in the final set of findings (see [A.1.13.2.10.2.4 - Publication Of Final Findings By Core GovOps](453f04da-9fcc-4a49-bd3e-374908d60319)). If the Agent does not, Core GovOps may propose an Atlas Edit Proposal to directly modify the Agent Artifact to address the deficiencies.
