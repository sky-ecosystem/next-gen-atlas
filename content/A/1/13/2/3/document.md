###### A.1.13.2.3 - Location Of Agent Artifacts [Core]

All Agent Artifacts, including new Agent Artifacts, must be recorded and publicly documented in the Agent Scope of the Sky Atlas: see [A.6 - The Agent Scope](4a08ca6c-e652-49e4-9b79-4831b20e600a).
