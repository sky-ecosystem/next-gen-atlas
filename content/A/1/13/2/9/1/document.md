###### A.1.13.2.9.1 - Standardized Executor Agent Sky Primitives [Core]

The following Sky Primitives are available for Executor Agents to Activate: all Genesis Primitives (the Agent Creation Primitive, the Agent Token Primitive, the Prime Transformation Primitive and the Executor Transformation Primitive), Operational Primitives (the Executor Accord Primitive, the Root Edit Primitive and the Light Agent Primitive) and the Executor’s own version of the Upkeep Primitive.
