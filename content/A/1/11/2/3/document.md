###### A.1.11.2.3 - Atlas Edit Proposal Template [Core]

All Atlas Edit Proposals must use the following template:

Preamble

AEP#: #
Author(s):
Contributors:
Status:
Date Proposed: <yyyy-mm-dd>
Date Ratified: <yyyy-mm-dd>
Forum URL:

Motivation

- Explain the motivation behind this AEP.

Edited Atlas Documents

- List the Atlas Documents that are being edited. Please include the Portal URL pointing to the specific Atlas Documents to facilitate identification.

Edit Pull Request

- Link to the Pull Request containing the proposed Atlas Edit. The Core Facilitator can create the Pull Request on request.
