###### A.1.1.2.1.0.3.4 - Precedent - Element Annotation [Annotation]

The element "precedent" refers to the established examples or standards set by documented Atlas Interpretations, which guide future decision-making and interpretations. Atlas Interpretations can be codified in the Facilitator Action Precedent Document Type. See [A.1.2.2.2.14 - The Facilitator Action Precedent Type](7323b3c5-ffd3-4f8f-af51-5d176531dd0f).
