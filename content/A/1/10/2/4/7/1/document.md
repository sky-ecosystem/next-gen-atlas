###### A.1.10.2.4.7.1 - Spell Crafting Workflow [Core]

In the Development Stage, the Spell Crafter writes the code for the Spell based on the contents of the Executive Sheet. The output of the Development Process is a Pull Request containing the code for the Spell that is ready for review. In performing the Development Stage, the Crafter takes the steps set forth in the checklists found in the Core Spell Crafter Mainnet Workflow, as specified in [A.1.10.2.5.1.3.2.0.6.1 - Registered Spell Checklists](93f5b36b-06a7-4282-9fd7-14e0cbafd08e).
