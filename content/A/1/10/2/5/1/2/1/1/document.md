###### A.1.10.2.5.1.2.1.1 - Agent Spell Reviewer Checklist [Core]

The Agent Spell Reviewer Checklist defines standardized steps for Agent Spell Reviewers. Beginning with the June 18, 2026 Executive Vote, a completed Agent Spell Reviewer Checklist must be included in the Spell review Pull Request prior to Spell handover for every Agent Spell. Beginning with the July 16, 2026 Executive Vote, each Agent Spell Reviewer, whether internal or external, must complete and include their own Agent Spell Reviewer Checklist.
