###### A.1.10.2.5.2.1.2 - Technical Scope Template Update Process [Core]

The Technical Scope Template is maintained by Core GovOps. All changes must be submitted as Pull Requests to the Ecosystem Operational References repository (https://github.com/soterlabs/ecosystem-operational-references) and require two (2) approvals, one (1) from each Core Spell Team, before they can be merged.
