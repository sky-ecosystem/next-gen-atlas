###### A.1.2.2.2.9 - The Element Annotation Directory Type [Type Specification]

[See below]

**Components**:

"Directory index": This Component should contain a list of the Element Annotation Documents contained within, with keys being the document identifiers and values being their document names.

**Doc Identifier Rules**:

Element Annotation Directory Documents must always be located at the .0.3 position of their Target Document.

**Additional Logic**:



**Type Category**:

Supporting Document

**Type Name**:

Element Annotation Directory

**Type Overview**:

The Element Annotation Directory Type is a directory used to list all of the Element Annotation Documents pertaining to a Target Document.
