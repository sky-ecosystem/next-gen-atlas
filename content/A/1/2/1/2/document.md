###### A.1.2.1.2 - Atlas Document Properties [Core]

All Atlas Documents have five (5) standard properties that provide key information about them. The subdocuments of this document specify the characteristics of each of the five (5) standard properties.

The standard properties of Atlas Documents should be listed in the following order:

1. Name
2. Version
3. Last Modified
4. Type
5. Components
