###### A.3.2.2.1.2.2.3.1 - Code Complexity Rating Maximum [Core]

The Code Complexity Rating Maximum $\text{CCRMax}$ is the maximum Code Complexity Rating for a protocol. The $\text{CCRMax}$ is set to `75`.
