###### A.3.2.2.1.2.3.1.2.3 - Calculate Capital Requirement Ratio Using F1 Piecewise Function [Core]

The Capital Requirement Ratio using the F1 Piecewise Function should then be calculated by inputting the F1 parameter calculated in [A.3.2.2.1.2.3.1.2.1 - Calculate F1 Parameter](fb47383f-e7f9-412b-893d-86e4c7b83def) into the Piecewise Function specified in [A.3.2.2.1.2.3.3.1 - Piecewise Function](7441253c-0030-4b8e-ac91-65046761aab6) using the F1 thresholds specified in [A.3.2.2.1.2.3.3.1.5.3 - Kink Threshold](c28b47a8-951b-4ca4-8501-bbd6d2279b74).
